#### GenAIOps - Demo Showcase

- Practical demonstration of GenAIOps in action

#### Shipping Code in the Agentic Era - Scenarios

- **Legacy workflow**: One person, one task, sequential execution.
- **AI-enabled workflow**: Multi-agent parallel execution with Git worktrees preventing conflicts.
- **Anthropic Agent Steams**: Context-sharing among agents for collaborative task completion.
- **Developer role evolution**: From coder to supervisor, managing orchestration across agents.

#### Shipping Code - Tools

- Git worktrees for isolation
- Agent supervisors for orchestration
- Custom toolchains for multi-repo management

#### Shipping Code - Deep Dive

- Single-agent drawbacks: limited context, sequential bottlenecks, stalled tasks
- Multi-agent benefits: parallel bug fixing, feature development, autonomous commits

#### Shipping Code - Demo

- Worktree setup → agent assignment → independent commits → supervisor merge

#### Shipping Code - Productivity Playbook

- Prompt engineering
- Context engineering
- Harnessing engineering
- Continuous learning

#### Production-Grade Multimodal GenAI on AWS

- **Legacy AI workflows**: Keyword search, siloed dashboards, fragmented data.
- **Modern AI stack**: Multimodal embeddings, unified semantic space, autonomous agents.

#### Production-Grade - GraphRAG for Enterprise Knowledge

#### Production-Grade - Multi-Agent Workflows

#### Production-Grade - Safe & Observable GenAI

- Guardrails for compliance and safety.
- Observability for monitoring workflows and embeddings.
- Scalable, serverless deployment.
